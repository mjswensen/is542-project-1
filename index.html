The Scala Programming Language

"Object-Oriented Meets Functional"

Introduction

Scala is a JVM-based language (i.e., compiles to Java bytecode) that fuses the features of modern programming languages to the ubiquity of Java-enabled platforms. From the official Scala website, "Scala is a concise, elegant, type-safe programming language that integrates object-oriented and functional features."

History

Scala began in 2001 as the brainchild of Martin Odersky, in Lausanne, Switzerland. A complete history of the language can be found at on the Scala website's history page.

Key Features

This document briefly overviews four of Scala's primary attractions and features:

  1. Interoperability with Java
  2. Functional Programming
  3. Flexibility with Object Inheritance
  4. Extensibility

Interoperability with Java

One of Scala's most powerful selling points is its compatibility with Java. Any Java code can be used within your Scala project! Because of Java's age and popularity, there are many high-quality Java libraries available. Scala can leverage these libraries.

Support for the .NET framework is also available.

Functional Programming

Scala introduces functional programming paradigms to the object-oriented world. Functions, therefore are first-class citizens in the Scala world and can receive the same treatments as any other object (e.g., be assigned to variables, be passed to other functions as parameters, be returned from other functions, etc.).

Flexibility with Object Inheritance

Scala supports inheritance the way a programmer familiar with the classical paradigms would expect. However, Scala also supports much more flexible approaches to code reuse. For example, Scala utilizes traits, or bits of functionality that can be mixed in to various classes.

Extensibility

Scala is extensible in that programmers can extend the language to meet the needs of their project. For example, any method can be used as an infix operator, like so: <object> <methodname> <otherobject>.

Conclusion

There are hundreds of other features of Scala that are worth mentioning. For a comprehensive look at the language, see the book written by Scala's creators: "Programming in Scala" by Martin Odersky, Lex Spoon, and Bill Venners.
